# A Simple Calculator made using Arduino

### Things Required:
1. Arduino Uno/Nano
2. 4*4 Matrix Keypad
3. 16*2 LCD with I2C Module

### Libraries used: 
1. [LiquidCrystal_I2C](https://github.com/johnrickman/LiquidCrystal_I2C) by Frank de Brabander
2. [Keypad](https://github.com/Chris--A/Keypad) by Mark Stanley
3. [BigNumber](https://github.com/nickgammon/BigNumber) by NickGammon
### Things It Can Do:
Addition, Subtraction, Multiplication and Division.

### Things It Can't Do: 
1. BODMAS.
2. Continuing calculation with the result (Too Lazy to add it).
